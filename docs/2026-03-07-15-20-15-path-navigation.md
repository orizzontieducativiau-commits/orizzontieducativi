# Path Navigation — Percorso visivo con icone cliccabili

**Data:** 2026-03-07
**Stato:** Completato

## Obiettivo

Creare una sezione di navigazione visiva ispirata a qatchup.com: una linea tratteggiata curva SVG con 10 icone circolari cliccabili posizionate lungo il percorso. Ogni icona rappresenta una tappa del "viaggio educativo" e porta alla sezione corrispondente con smooth scroll.

La linea deve collegare visivamente la fine della Hero ("Educativi Australia.") all'inizio della sezione About ("standard richiesti"), creando continuità visiva.

## Approccio tecnico

### 1. Nuovo componente `src/components/PathNavigation.tsx`
- **Client Component** (`"use client"`) per gestire hover states e smooth scroll
- Array di dati con 10 item: `{ label, href, icon, color }`
- Linea curva SVG tratteggiata (`stroke-dasharray`) in grigio molto chiaro
- Icone Lucide React in cerchi con sfondo pastello leggero
- Hover: leggero scale up + ombra soft
- Focus states per accessibilità (aria-label su ogni link)
- **Desktop**: percorso curvo che serpeggia orizzontalmente, icone posizionate lungo la curva
- **Mobile**: layout orizzontale scrollabile o griglia semplificata
